# 便秘AI 待合アンケートと医師問診の役割分担

作成日: 2026-04-28

目的:

- 待合で保護者が短時間に回答できる項目と、医師が診察中に確認すべき項目を分ける。
- 全例・毎回聞く項目を最小化し、必要時だけAIが追加質問する設計にする。
- 2-3歳モビコール導入・維持期の日誌、医師サマリー、診察前問診を接続する。

## 基本方針

待合アンケートは「短い・迷わない・医師の診察前に役立つ」ことを優先する。

医師問診は、背景の解釈、診察所見との統合、治療方針の判断が必要な項目を扱う。

AIは、アンケート回答と日誌から不足情報を見つけ、必要な場合だけ追加質問を出す。ただし、診断、処方変更、治療中止判断は行わない。

## 1. 待合で全例・毎回聞く項目

目安: 1-2分で回答できる量。

| 項目 | 回答形式 | 理由 |
|---|---|---|
| 最後に便が出た日 | 今日 / 昨日 / 2-3日前 / 4日以上前 / 不明 | 4日以上の無排便を拾う |
| 最近の排便回数 | 1日複数回 / 1日1回 / 2-3日に1回 / 4日以上あく / 不明 | 治療反応と再燃評価 |
| 便の硬さ | 硬い / 普通 / ゆるい / 水様 / 不明 | モビコール調整相談の材料 |
| 排便時に痛がるか | ない / 少し / 強い / 泣く / 不明 | 排便がまんの悪循環評価 |
| 排便をがまんする様子 | ない / ある / 強い / 不明 | 行動面の評価 |
| 血がつくこと | ない / 紙に少量 / 便に混じる / 不明 | 硬便・裂肛などの相談材料 |
| 嘔吐・強い腹痛・強い腹部膨満 | ない / ある / 不明 | 早期対応が必要な症状確認 |
| モビコールを飲めているか | 処方通り / 時々忘れる / 飲めない / 中止中 / 該当なし | 効果判定の前提 |

## 2. 待合で初回または久しぶりの受診時に聞く項目

毎回ではなく、初回、再開時、久しぶりの受診、症状悪化時に聞く。

| 項目 | 回答形式 | 理由 |
|---|---|---|
| 便秘が始まった時期 | 0歳台 / 1歳頃 / 2歳以降 / 不明 | 慢性経過の把握 |
| これまでの浣腸歴 | なし / 時々 / 繰り返し / 不明 | 重症度・過去対応 |
| 便でトイレが詰まること | なし / あり / 不明 | 大量便・便貯留の手がかり |
| 1回の排便量 | 少ない / 普通 / 多い / 非常に多い / 不明 | 便貯留や排便間隔との関連 |
| 便失禁・下着汚れ | なし / あり / 不明 | 便塞栓・再燃の相談材料 |
| 園での排便困難 | なし / あり / 園に通っていない / 不明 | 生活上の困りごと |
| 薬が飲みにくいか | なし / あり / 不明 | 治療継続性 |

## 3. 初回登録で聞く背景情報

背景情報は毎回聞かない。初回登録時に聞き、変更があれば更新する。

| 項目 | 例 |
|---|---|
| 周産期歴 | 低出生体重児、極低出生体重児、早産、NICU歴 |
| 基礎疾患・既往歴 | 神経疾患、心疾患、消化器疾患など |
| 発達・行動面 | 発達障害の診断あり、未診断だが多動傾向、トイレ協力困難 |
| アレルギー・併存疾患 | アレルギー性鼻炎、食物アレルギー疑いなど |
| 定期内服薬 | オロパタジンなど |
| 家族歴 | 兄弟、親の便秘治療歴 |

注意:

- AIは、背景情報と便秘の因果関係を断定しない。
- 未診断の発達特性は「診断」ではなく「診察中の様子」「保護者の困りごと」として扱う。

## 4. 条件付きでAIが追加質問する項目

全例に毎回聞くと負担が大きい項目は、回答内容に応じてAIが追加で聞く。

| 条件 | 追加質問 |
|---|---|
| 4日以上排便なし | 「お腹の張り、腹痛、嘔吐はありますか」 |
| 硬便 + 痛みあり | 「血がつくことはありますか」「排便を怖がる様子はありますか」 |
| 無排便後に排便あり | 「1回量は多かったですか」「トイレが詰まるほどでしたか」 |
| モビコールを飲めていない | 「味、量、タイミングなど、飲めない理由は何ですか」 |
| 中止中・飲み忘れあり | 「中止や飲み忘れの後、便が硬くなったり間隔があいたりしましたか」 |
| 園で排便困難あり | 「園では便意をがまんする様子がありますか」「排便に時間がかかりますか」 |

## 5. 医師が診察中に確認する項目

以下は、アンケートだけで全例毎回聞き切るより、医師が必要に応じて確認する方がよい。

- 便塞栓の有無の評価。
- 腹部診察、肛門周囲の評価、直腸肛門指診の要否。
- 出血の程度や持続、診察所見との照合。
- 嘔吐、強い腹痛、強い腹部膨満の医学的評価。
- 基礎疾患やレッドフラッグの鑑別。
- モビコールの継続、減量、中止、他薬追加の判断。
- トイレトレーニング開始・再開のタイミング。
- 専門医療機関紹介の要否。

## 6. 診察前サマリーの形

待合アンケートが終わった時点で、AIは医師に以下の短いサマリーを出す。

```text
【診察前 便秘サマリー】

最終排便: 2-3日前
最近の排便頻度: 2-3日に1回
便性状: 硬い
排便時痛: 強い
がまん: あり
出血: 紙に少量
嘔吐/強い腹痛/強い腹部膨満: なし
モビコール: 処方通り内服

追加確認候補:
- 硬便 + 痛み + 出血あり。裂肛や痛みによる排便がまんについて診察で確認。
- 4日以上の無排便は今回なし。
- 処方変更判断はAIでは行っていない。
```

## 7. 実装上の優先順位

最初に実装する:

- 待合で全例・毎回聞く8項目。
- 初回登録の背景情報。
- 4日以上無排便、嘔吐、強い腹痛・腹部膨満の早期相談フラグ。
- 診察前サマリー。

次に実装する:

- 条件付き追加質問。
- 園での排便困難。
- トイレ詰まり、大量便、便失禁の詳細。
- 減量・中止・飲み忘れ後の再燃チェック。
