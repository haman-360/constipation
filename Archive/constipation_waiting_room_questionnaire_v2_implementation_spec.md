# 2-3歳便秘AI 待合アンケート v2 実装仕様

作成日: 2026-04-28

参照:

- `constipation_waiting_room_questionnaire_v2.md`
- `constipation_waiting_room_questionnaire_v2_revision_notes.md`

目的:

- v2アンケートを、待合タブレットまたはスマートフォン画面に実装しやすい形へ落とす。
- 各質問の入力形式、表示条件、医師サマリーへの出力先を明確にする。
- AIが診断、処方変更、治療中止判断をしない境界を実装側でも保つ。

## 1. 画面構成

### 画面A. 基本入力

待合で最初に表示する。目安は1-2分。

- Q1 最終排便
- Q2 最近の排便頻度
- Q3 便の硬さ
- Q4 排便時痛
- Q5 排便がまん
- Q6 便秘薬の内服状況

### 画面B. 自動展開される症状確認

画面Aの回答に応じて表示する。  
全例に毎回確認してもよいが、短縮UIでは条件付き表示を基本にする。

- Q7 出血
- Q8 便失禁・下着汚れ
- Q9 腹痛・腹部膨満
- Q10 嘔吐
- Q11 食欲・機嫌
- Q12 1回排便量
- Q13 便秘薬が飲みにくい理由
- Q14 飲み忘れ・中止・少なめ後の変化
- Q15 トイレ詰まり
- Q16 園・外出先での排便困難

### 画面C. 初回・久しぶり受診時

初回、久しぶりの受診、症状悪化時に表示する。

- Q17 便秘開始時期
- Q18 浣腸歴
- Q19 受診中断歴
- Q20 家族歴

### 画面D. 初回登録・背景情報

毎回は聞かず、初回登録時または変更時だけ確認する。

- B1 新生児期の通院歴
- B2 診断・通院中の病気
- B3 発達・行動面
- B4 定期内服薬

## 3. 表示条件ロジック

### 4日以上無排便

条件:

- q1_last_bowel_movement = 4日以上前
- または q2_bowel_frequency = 4日以上あくことがある

表示:

- q9_abdominal_symptom
- q10_vomiting
- q11_appetite_mood

サマリー強調:

- 4日以上無排便または排便間隔延長あり。

### 硬便・痛み・がまん

条件:

- q3_stool_consistency = 硬い
- または q4_pain = 強く痛がる / 泣く、またはとても嫌がる
- または q5_withholding = ある / 強くある

表示:

- q7_blood
- q8_soiling
- q12_stool_amount

サマリー強調:

- 硬便、排便時痛、がまん、出血の組み合わせを医師確認候補に出す。

### 水様便

条件:

- q3_stool_consistency = 水のよう

表示:

- q9_abdominal_symptom
- q10_vomiting
- q11_appetite_mood

サマリー強調:

- 水様便と全身状態の変化を並べて出す。

### 便失禁・下着汚れ

条件:

- q8_soiling = ときどきある / よくある

表示:

- q9_abdominal_symptom
- q12_stool_amount

サマリー強調:

- 便失禁・下着汚れあり。
- 便塞栓の有無はAI判断しない。

### 内服が少なめ・飲み忘れ・中止

条件:

- q6_med_status = 調節してよいと言われたが、最近は少なめになっている
- q6_med_status = ときどき忘れる
- q6_med_status = 今は中止している
- q6_med_adherence_flags に ときどき忘れる が含まれる

表示:

- q14_change_after_less_med

サマリー強調:

- 実際の内服が少なめ、飲み忘れ、または中止中。
- その後の硬便、排便間隔延長、痛みを併記する。

### 内服困難

条件:

- q6_med_status = 飲みにくくて残る
- q6_med_adherence_flags に 飲みにくくて残る が含まれる

表示:

- q13_med_difficulty_reason

サマリー強調:

- 内服困難あり。
- 理由を味、量、タイミング、子どもの拒否などに分けて出す。

### 医師指示範囲内の自己調節

条件:

- q6_med_status = 医師から調節してよいと言われており、便の様子を見ながら調節している

表示:

- 最近少なめ、飲み忘れ、中止、保護者の心配がなければq14は必須にしない。

サマリー出力:

```text
便秘薬の内服:
- 状況: 医師から調節してよいと言われており、便の様子を見ながら調節している
- 医師から調節可の説明: あり
- 実際の内服: 医師指示範囲内で調節
- 最近少なめになっている: なし / 不明
```

### 大量便・トイレ詰まり

条件:

- q12_stool_amount = とても多い
- かつトイレで排便している、または医師が確認したい場合

表示:

- q15_toilet_clogging

サマリー強調:

- 大量便エピソードあり。
- トイレ排便なしの場合、トイレ詰まりは「トイレ排便なし」として扱う。

## 4. サマリー生成ルール

### 便秘薬の内服

q6_med_statusから以下を生成する。

| q6回答 | 医師から調節可の説明 | 実際の内服 | 最近少なめ |
|---|---|---|---|
| 医師に言われた量で飲んでいる | なし | 医師指示量 | なし |
| 医師から調節してよいと言われており、便の様子を見ながら調節している | あり | 医師指示範囲内で調節 | なしまたは不明 |
| 調節してよいと言われたが、最近は少なめになっている | あり | 最近少なめ | あり |
| ときどき忘れる | 不明 | 飲み忘れあり | 不明 |
| 飲みにくくて残る | 不明 | 内服困難あり | 不明 |
| 今は中止している | 不明 | 中止中 | 不明 |
| 便秘薬は使っていない | なし | 便秘薬なし | なし |
| わからない | 不明 | 不明 | 不明 |

### AI追加確認候補

以下に該当する場合、サマリーに短く出す。

- 4日以上無排便または排便間隔延長。
- 硬便 + 排便時痛。
- 硬便 + 出血。
- 排便がまんあり。
- 便失禁・下着汚れあり。
- 強い腹痛・腹部膨満。
- 嘔吐あり。
- 食欲低下または機嫌不良。
- 内服困難あり。
- 最近少なめ、飲み忘れ、中止後に硬便、排便間隔延長、痛みがある。

### AIが判断しない項目

必ずサマリー末尾に残す。

- 診断
- 便塞栓の有無
- モビコールなどの処方量変更
- 治療中止可否
- 専門紹介の要否

## 5. 保護者向け文言ルール

- 「便塞栓」「裂肛」などの診断名は、保護者画面では原則使わない。
- 「薬を増やす」「薬を減らす」「中止してよい」とは表示しない。
- 「医師に相談する材料として記録します」「診察前サマリーに表示します」と表現する。
- 「わからない」を選んでも先に進める。
- 2-3歳ではおむつ排便があるため、トイレ前提の表現を避ける。

## 6. 最小実装の優先順位

### 必須

- Q1-Q6
- 4日以上無排便時のQ9-Q11
- 硬便・痛み・がまん時のQ7、Q8、Q12
- Q6に応じたQ13、Q14
- 診察前サマリー

### 次点

- Q15 トイレ詰まり
- Q16 園・外出先での困りごと
- Q17-Q20 初回・久しぶり受診時項目
- B1-B4 背景情報

### 後回しでもよい

- 自由記載の詳細分類。
- 医師側で表示条件をカスタマイズする管理画面。
- 日誌データとの自動統合。
