# 2-3歳便秘AI 待合アンケート 医師レビュー用要約

作成日: 2026-04-28

## 目的

2-3歳の便秘診療で、診察前に保護者から最近のうんちの様子とお薬の状況を短時間で入力してもらい、医師が診察前に状態を把握しやすくする。

対象は、モビコールなどの便秘薬導入前、導入期、維持期、減量・中止後観察を含む。

## このAIがすること

- 保護者の回答を構造化する。
- 最終排便、排便頻度、便の硬さ、痛み、がまん、お薬の状況を整理する。
- 必要時に、腹痛・お腹の張り、嘔吐、食欲・機嫌、出血、便が少しつく様子を追加確認する。
- 医師向けに診察前ミニサマリーを作成する。
- 強い腹痛、嘔吐、食欲・機嫌の変化がある場合、保護者に受付または医療スタッフへ伝えるよう案内する。

## このAIがしないこと

- 診断。
- 便塞栓の有無判断。
- モビコールなどの処方量変更。
- 薬を増やす、減らす、中止する指示。
- 治療中止可否の判断。
- 専門紹介の要否判断。
- 先天性疾患の有無判断。

## MVPで聞く基本6問

| 項目 | 保護者向け質問 |
|---|---|
| 最終排便 | 最後にうんちが出たのはいつですか？ |
| 排便頻度 | 最近、うんちはどれくらいの間隔で出ていますか？ |
| 便の硬さ | 最近のうんちの硬さはどれに近いですか？ |
| 排便時痛 | うんちをするとき、痛がることはありますか？ |
| がまん | うんちをがまんしているように見えることはありますか？ |
| 便秘薬 | 便秘のお薬は、今どのように飲んでいますか？ |

## 条件付き追加確認

| 条件 | 追加で確認すること |
|---|---|
| 4日以上うんちが出ていない、または4日以上あくことがある | お腹の張り・腹痛、嘔吐、食欲・機嫌 |
| 水のような便 | お腹の張り・腹痛、嘔吐、食欲・機嫌 |
| 硬い便、強い痛み、泣く・とても嫌がる、がまんあり | 血がつくこと、パンツやおむつに少し便がつくこと |
| お薬が飲みにくくて残る | 味、量、飲ませるタイミング、子どもの拒否など |
| お薬が少なめ、飲み忘れ、中止中 | その後の硬さ、間隔、痛みの変化 |

## 便秘薬の自己調節の扱い

維持期では、医師が「便の様子を見ながら調節してよい」と説明している場合がある。

そのため、AIは以下を分けて記録する。

- 医師に言われた量で飲んでいる。
- 医師から調節してよいと言われており、便の様子を見ながら調節している。
- 調節してよいと言われたが、最近は少なめになっている。
- ときどき忘れる。
- 飲みにくくて残る。
- 今は中止している。
- 便秘薬は使っていない。
- わからない。

医師指示範囲内の安定した自己調節だけでは、薬を少なめにした後の変化を必須質問にしない。  
最近少なめ、飲み忘れ、中止、飲みにくさがある場合に追加確認する。

## 保護者向け安全表示

以下がある場合は、終了画面で受付または医療スタッフにも伝えるよう表示する。

- 強い腹痛・お腹の張り。
- 嘔吐。
- 食欲が少ない。
- 機嫌が悪い。
- 食欲も機嫌も気になる。

表示文:

```text
強い腹痛、吐いた、食欲や機嫌がいつもと違う場合は、入力だけでなく受付または医療スタッフにもお伝えください。
```

## 医師向けミニサマリー例

```text
【診察前 便秘ミニサマリー】

最終排便: 4日以上前
最近の排便頻度: 4日以上あくことがある
うんちの硬さ: 硬い
排便時痛: 強く痛がる
がまん: ある

お薬:
- 状況: 先生に言われた量で飲んでいる
- 補足: なし
- 少なめ・飲み忘れ・中止後の変化: 未確認
- 飲みにくさ: 未確認

追加確認:
- お腹の張り・腹痛: 少しある
- 嘔吐: ない
- 食欲・機嫌: いつも通り
- 血がつく: 紙に少しつく
- パンツやおむつに少し便がつく: ない

AI追加確認候補:
- 4日以上うんちが出ていないため、お腹の張り・腹痛、嘔吐、食欲・機嫌を診察で確認。
- 硬い便、痛み、がまんの組み合わせを確認。

AIが判断していないこと:
- 診断
- 便塞栓の有無
- モビコールなどの処方量変更
- 治療中止可否
- 専門紹介の要否
```

## 医師レビューで確認したい点

- 基本6問で、診察前の状態把握として不足がないか。
- 4日以上無排便、水様便、強い痛み、がまんに対する追加確認が妥当か。
- 薬の自己調節、少なめ、飲み忘れ、飲みにくさの分け方が臨床実態に合うか。
- 保護者向け安全表示の条件と文言が適切か。
- 医師向けミニサマリーに、不要な判断や踏み込みすぎた表現がないか。

## 初回プロトタイプでは扱わない項目

- 周産期・新生児期の詳細。
- 基礎疾患、発達・行動面、定期薬。
- 家族歴。
- トイレ詰まり。
- 園・外出先での排便困難。
- 浣腸歴、受診中断歴、便秘開始時期。

これらは、初回レビュー後に必要性の高いものから追加する。
