# 便秘日誌・医師サマリー補遺: 抜けやすい重要確認項目

作成日: 2026-04-28

目的:

- 小児便秘診療でよく確認するが、日誌・問診から抜けやすい項目を明文化する。
- 2-3歳モビコール導入・維持期の日誌設計と医師サマリーに追加反映する。
- AIが診断や処方変更を行わず、医師が判断しやすい観察情報として整理する。

## 追加すべき重要項目

| 項目 | 入力形式 | 医師サマリーでの意味 |
|---|---|---|
| 出血の有無 | なし / 紙に少量 / 便に混じる / 不明 | 硬便、裂肛、医師相談材料 |
| 便でトイレが詰まるか | なし / あり / 不明 | 大量便、便貯留、慢性経過の手がかり |
| 1回の排便量 | 少ない / 普通 / 多い / 非常に多い / 不明 | 排便間隔、便貯留、便塞栓疑いの相談材料 |
| 嘔吐 | なし / あり / 不明 | 早期相談が必要な合併症状の確認 |
| その他の合併症状 | 腹痛 / 腹部膨満 / 食欲低下 / 機嫌不良 / 発熱など | 全身状態、安全面の評価 |
| 排便時痛 | なし / 少し / 強い / 泣く / 不明 | 排便がまんの悪循環評価 |
| 排便間隔が4日以上あく | なし / あり / 不明 | 再燃、便貯留、受診相談材料 |

## AIの聞き返しルール

日誌に以下の情報がない場合、AIは保護者に短く聞き返す。

- 4日以上排便がない、または排便間隔が不明な場合: 「4日以上出ないことはありますか」
- 硬便、痛み、がまんがある場合: 「血がついたことはありますか」
- 無排便が続いた後に排便があった場合: 「1回量がかなり多い、またはトイレが詰まるほどでしたか」
- 腹痛や腹部膨満がある場合: 「吐いたことはありますか」
- トイレに長時間座る、排便に時間がかかる場合: 「出る量は少量ですか、それとも一度に多く出ますか」

## 医師サマリーへの追加欄

既存の医師サマリーテンプレート「現在の排便状況」に以下を追加する。

```text
- 最長無排便日数:
- 4日以上の無排便: なし / あり（回数）
- 1回排便量: 少ない / 普通 / 多い / 非常に多い / 不明
- 便によるトイレ詰まり: なし / あり / 不明
- 出血: なし / 紙に少量 / 便に混じる / 不明
- 合併症状: 嘔吐 / 腹痛 / 腹部膨満 / 食欲低下 / 機嫌不良 / 発熱 / なし / 不明
```

## 日誌画面への反映

毎日必須に近い項目:

- 排便の有無
- 排便回数
- 便の硬さ
- 1回の排便量
- 排便時痛
- 排便がまん
- 腹痛・腹部膨満
- 嘔吐
- モビコール内服状況

詳細入力または条件付き展開:

- 出血
- 便でトイレが詰まる
- 便失禁・下着汚れ
- トイレ・おまるに座れたか
- 浣腸・坐薬・他薬剤の使用
- 発熱や感染症

条件付きで詳細入力を自動展開する場面:

- 排便間隔が4日以上
- 排便時に強く痛がる、泣く
- 硬便が続く
- 便失禁または下着汚れがある
- 腹部膨満、強い腹痛、嘔吐がある
- 無排便後に大量便が出た

## AI判定ラベルへの追加

| ラベル | 条件の例 | AIの返答方針 |
|---|---|---|
| 便貯留の相談材料あり | 4日以上の無排便、大量便、トイレ詰まり、便失禁、腹部膨満が組み合わさる | 医師に相談するため、直近の日誌を整理 |
| 硬便・裂肛疑いの相談材料あり | 硬便、排便時痛、出血が組み合わさる | 出血の程度を整理し、医師相談を促す |
| 要早期相談 | 嘔吐、強い腹痛、著しい腹部膨満、全身状態低下 | 速やかな医療相談を促す |

## 保護者向け応答例

```text
4日以上排便があいた記録があります。
便の硬さ、排便時の痛み、出血、1回量が多いか、トイレが詰まるほどだったかを確認しておくと、受診時に医師が経過を判断しやすくなります。
薬の量の変更はAIでは判断できないため、日誌をまとめて処方医に相談してください。
```

```text
硬い便と排便時痛が記録されています。
血がついたことがあるか、排便をがまんする様子があるかも確認しておきましょう。
痛みが続く時期は、トイレトレーニングを無理に進めず、次回受診時に相談する材料として記録します。
```
