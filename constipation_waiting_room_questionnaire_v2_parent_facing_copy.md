# 2-3歳便秘AI 待合アンケート v2 保護者向け表示文言

作成日: 2026-04-28

参照:

- `constipation_waiting_room_questionnaire_v2.md`
- `constipation_waiting_room_questionnaire_v2_implementation_spec.md`

目的:

- 待合で保護者が実際に読む画面文言を作る。
- 医療者向けの言葉を避け、短く、迷わず、責められている感じが出ない表現にする。
- AIが診断や薬の量の判断をしないことを、必要な場所だけ自然に伝える。

## 1. 全体の言葉づかい

基本方針:

- 「排便」より「うんち」を使う。
- 「便性状」より「うんちの硬さ」を使う。
- 「内服」より「飲んでいる」「飲めている」を使う。
- 「便失禁」より「パンツやおむつに少しつく」を使う。
- 「腹部膨満」より「お腹の張り」を使う。
- 「自己調節」より「先生から調節してよいと言われている」を使う。

避ける表現:

- 「便塞栓」
- 「裂肛」
- 「治療不十分」
- 「コンプライアンス不良」
- 「怠薬」
- 「自己判断で減量」

使う表現:

- 「診察で確認しやすくするため」
- 「わかる範囲で大丈夫です」
- 「近いものを選んでください」
- 「薬の量は、この画面では判断しません」

## 2. 開始画面

### タイトル

```text
うんちの様子を教えてください
```

### 説明文

```text
診察の前に、最近のうんちの様子を確認します。
わかる範囲で、近いものを選んでください。
薬の量をこの画面で決めることはありません。
```

### 開始ボタン

```text
はじめる
```

## 7. 自動展開時のつなぎ文

### 4日以上うんちが出ていないとき

```text
うんちの間隔があいているため、体調について少し確認します。
```

### 硬いうんち・痛み・がまんがあるとき

```text
うんちが硬い、痛がる、がまんする様子があるため、関連することを少し確認します。
```

### お薬が少なめ・飲み忘れ・中止のとき

```text
お薬を飲まなかったり少なめになったりした後の変化を確認します。
薬の量は診察で相談してください。
```

### お薬が飲みにくいとき

```text
飲みにくさの理由を確認します。
診察で相談しやすくするための質問です。
```

### トイレでうんちをすることに関する質問の前

```text
トイレでうんちをしている場合だけ答えてください。
まだの場合は「まだトイレではうんちをしていない」を選んでください。
```

## 9. 注意表示

通常は画面下部または終了画面に短く表示する。

```text
この画面では、診断や薬の量の変更は行いません。
気になる症状がある場合は、診察時に医師へお伝えください。
```

強い腹痛、嘔吐、食欲・機嫌の低下が選ばれた場合は、終了画面で以下を追加表示する。

```text
お腹の強い症状や吐き気、食欲・機嫌の変化がある場合は、受付または医療スタッフにもお伝えください。
```
