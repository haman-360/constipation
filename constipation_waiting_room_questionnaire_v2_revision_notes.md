# 待合アンケート v2 修正メモ

作成日: 2026-04-28

目的:

- 慢性期・維持期のモビコール自己調節、2-3歳のおむつ排便、背景情報の聞き方を実運用に寄せる。

## 1. Q6 モビコール内服状況の修正

### 課題

v1:

```text
Q6. モビコールなどの便秘薬は飲めていますか？
- 処方どおり飲めている
- ときどき忘れる
- 飲みにくくて残る
- 今は中止している
- 便秘薬は使っていない
- わからない
```

慢性期・維持期では、医師が「便の性状や頻度を見ながら調節してよい」と伝えている場合がある。

一方で、調節可と伝えると実際には内服量が少なめになりやすい。したがって、単に「処方どおり飲めている」だけでは、医師の指示内の調節なのか、保護者判断で減っているのかが分からない。

### v2案

```text
Q6. モビコールなどの便秘薬は、今どのように飲んでいますか？

- 医師に言われた量で飲んでいる
- 医師から調節してよいと言われており、便の様子を見ながら調節している
- 調節してよいと言われたが、最近は少なめになっている
- ときどき忘れる
- 飲みにくくて残る
- 今は中止している
- 便秘薬は使っていない
- わからない
```

### 医師サマリーでの表示

```text
便秘薬の内服:
- 医師指示量で内服
- 医師指示範囲内で自己調節
- 調節可と説明あり、最近は少なめ
- 飲み忘れあり
- 内服困難あり
- 中止中
- 便秘薬なし
- 不明
```

### AIの扱い

- AIは「調節してよい」「減らしてよい」「増やしてよい」とは判断しない。
- 自己調節は、医師が許可した調節なのか、実質的な減量なのかを分けて記録する。
- 「少なめになっている」場合は、排便回数、便性状、排便時痛、排便間隔、再燃兆候と合わせて医師に提示する。

### 治療方針についての現時点の整理

この設計では、「固定量がよいか」「便性状に応じた調節がよいか」はAIが決めない。

ただし運用上は、以下のどちらの方針でも使えるようにする。

```text
A. 医師指示量を維持する方針
  -> Q6で「医師に言われた量で飲んでいる」を選択

B. 医師が調節範囲を伝える方針
  -> Q6で「医師から調節してよいと言われており、便の様子を見ながら調節している」を選択
  -> 実際に少なめになりがちな場合は「最近は少なめになっている」を選択
```

今後、院内ルールとして決めるべきこと:

- 調節可とする場合、何を目安に調節してよいと説明するか。
- 何日出なければ自己調節ではなく受診相談にするか。
- 減量しすぎを拾うため、再診時に実際の内服量を必ず確認するか。
- 日誌上で、処方量と実際の内服量を別々に記録するか。

## 2. Q15 トイレ詰まりの扱い

### 課題

2-3歳では、おむつ排便が多い。  
「便が大きくてトイレが詰まったことはありますか」は、自分でトイレ排便ができる児向けの質問であり、全例・毎回には不要。

### v2案

Q15は全例には表示しない。条件付き質問にする。

表示条件:

- トイレで排便している
- 年齢が上がってきた
- 便量が「とても多い」
- 医師が大量便エピソードを確認したい場合

質問文:

```text
Q15. 自分でトイレにうんちをする場合、便が大きくてトイレが詰まったことはありますか？

- ない
- ある
- まだトイレではうんちをしていない
- わからない
```

2-3歳版では、より優先する質問は以下。

```text
Q12. 1回のうんちの量は多いですか？
- 普通くらい
- 少ない
- 多い
- とても多い
- わからない
```

## 3. B1 周産期歴の聞き方

### 課題

「低出生体重児」「1500g未満」「早産」「NICU入院」は医学的には有用だが、保護者が待合で選びにくい場合がある。

### v2案

```text
B1. 生後、産婦人科や新生児科に通院したことはありますか？

- ない
- ある
- わからない
```

「ある」の場合だけ、可能なら自由記載または追加選択。
目的は詳細な周産期分類ではなく、周産期・新生児期に通常と異なる経過があり、先天性疾患などを診察で念頭に置く必要があるかを拾うこと。

追加候補:

- 小さく生まれた
- 早く生まれた
- NICUに入院した
- その他
- わからない

医師サマリーでは、以下のように表示する。

```text
周産期・新生児期:
- 生後に産婦人科/新生児科通院歴あり
- 詳細: 自由記載または選択内容
```

## 4. B4 定期薬の聞き方

### 課題

2-3歳ではアレルギー性鼻炎は多くない。  
B2で診断・通院中の病気を聞くため、B4では薬剤に絞った方が簡潔。

### v2案

```text
B4. 定期的に飲んでいる薬はありますか？

- なし
- あり
- わからない
```

「あり」の場合は薬剤名を自由記載。

医師サマリーでは、診断名と薬剤名を分ける。

```text
基礎疾患・通院中の病気:
- B2の回答

定期内服薬:
- B4の回答
```

## 6. v2追加修正メモ

2026-04-28追記。

### 表示方針

v2本文では「全例・毎回の基本質問」と「分岐表示」が混在すると実装時に迷うため、以下の扱いに整理した。

- 基本質問は全例・毎回の回答対象。
- 短縮UIではQ1-Q6を先に出し、Q7-Q11は回答内容に応じて自動展開してよい。
- 本文中の「分岐」は、短縮UIでの自動展開条件、または医師サマリーで強調する条件を意味する。

### Q6とQ14

Q6は、今の内服状況に一番近いものを1つ選ぶ形式を基本にする。  
ただし「ときどき忘れる」「飲みにくくて残る」は、他の状況と併存しやすいため追加チェックとしても記録できる設計にした。

Q14は、安定した医師指示範囲内の自己調節だけでは必須表示にしない。  
最近の実際量が少なめ、飲み忘れ、中止、または保護者が変化を気にしている場合に表示する。

### 医師サマリー

「医師指示範囲内の調節: あり」と「最近少なめ」を並べると、少なめ内服まで指示範囲内に見える恐れがある。  
そのため、サマリーでは以下の2層に分ける。

```text
- 医師から調節可の説明: あり / なし / 不明
- 実際の内服: 医師指示量 / 医師指示範囲内で調節 / 最近少なめ / 飲み忘れあり / 中止中 / 不明
```
