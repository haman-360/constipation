参照:

目的:

- 保護者向け表示文言を、実際の画面遷移として確認する。
- 条件付き質問が多すぎないか、自然な順番で出るかを確認する。
- 医師サマリーに必要な情報が、待合入力で拾えるかを確認する。

## 1. 共通開始画面

```text
うんちの様子を教えてください

診察の前に、最近のうんちの様子を確認します。
わかる範囲で、近いものを選んでください。
薬の量をこの画面で決めることはありません。

[はじめる]
```

## 2. フロー例A: 安定維持例

想定:

- 今日うんちが出ている。
- 1日1回くらい。
- 普通便。
- 痛み、がまん、出血、便汚れなし。
- 先生に言われた量で便秘薬を飲んでいる。

### 表示される質問

#### A-1. 最終排便

```text
最後にうんちが出たのはいつですか？

だいたいで大丈夫です。

- 今日
- 昨日
- 2-3日前
- 4日以上前
- わからない
```

回答例:

```text
今日
```

#### A-2. 最近の間隔

```text
最近、うんちはどれくらいの間隔で出ていますか？

ここ1-2週間くらいの様子で選んでください。

- 1日に2回以上
- 1日1回くらい
- 2-3日に1回くらい
- 4日以上あくことがある
- わからない
```

回答例:

```text
1日1回くらい
```

#### A-3. 硬さ

```text
最近のうんちの硬さはどれに近いですか？

いつも同じでない場合は、気になるものを選んでください。

- 硬い
- 普通
- やわらかい
- 水のよう
- わからない
```

回答例:

```text
普通
```

#### A-4. 痛み

```text
うんちをするとき、痛がることはありますか？

泣く、こわがる、強く嫌がる様子も含めてください。

- 痛がらない
- 少し痛がる
- 強く痛がる
- 泣く、またはとても嫌がる
- わからない
```

回答例:

```text
痛がらない
```

#### A-5. がまん

```text
うんちをがまんしているように見えることはありますか？

足を閉じる、隠れる、体をこわばらせる、トイレやおむつ替えを嫌がる様子などです。

- ない
- ある
- 強くある
- わからない
```

回答例:

```text
ない
```

#### A-6. お薬

```text
便秘のお薬は、今どのように飲んでいますか？

一番近いものを選んでください。飲み忘れや飲みにくさがある場合は、あとで追加で選べます。

- 先生に言われた量で飲んでいる
- 先生から調節してよいと言われていて、うんちの様子を見ながら調節している
- 調節してよいと言われたが、最近は少なめになっている
- ときどき忘れる
- 飲みにくくて残る
- 今は中止している
- 便秘のお薬は使っていない
- わからない
```

回答例:

```text
先生に言われた量で飲んでいる
```

### この例では追加表示しない質問

- Q8 パンツやおむつの便汚れ
- Q9 お腹の張り・腹痛

### 終了画面

```text
入力ありがとうございました

回答内容を診察前の確認用にまとめます。

```

## 6. フロー確認メモ

この画面フローで確認したい点:

- 安定例ではQ1-Q6だけで終われるため、待合負担は小さい。
- 4日以上うんちが出ていない例では、安全面のQ9-Q11が自然に追加される。
- 硬便、痛み、がまんがある例では、出血、便汚れ、1回量が拾える。
- 薬が少なめになっている例では、AIが薬の量を判断せず、変化だけを医師へ渡せる。
- 飲みにくさがある例では、味、量、タイミング、子どもの拒否を分けて拾える。

残る検討:

- Q8「パンツやおむつに少しつく」は、硬便だけで出すか、痛み・がまんがある時に限定するか。
- Q9-Q11は全例に毎回出す運用にするか、短縮UIでは4日以上無排便や水様便などに限定するか。
- 医師サマリーで「未確認」を表示するか、未表示項目は省略するか。
