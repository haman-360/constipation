# 0-1歳向け便秘問診プロトタイプ

## 目的

0-1歳の便秘相談で、保護者が待合で入力しやすく、医師が診察前に確認すべき情報を短時間で把握できる問診プロトタイプを作る。

この文書は医療者レビュー用のたたき台であり、診断、処方、紹介判断を自動化するものではない。

## 対象

- 0歳0か月から1歳11か月
- 便が少ない、硬い、痛がる、出しにくそう、薬や処置を使っている乳幼児
- 初診、再診の両方

## 設計方針

- 患者画面では、生年月日や年齢を自由入力させない。
- 年齢は `patients.birth_date` と受診日から医師側で計算する。
- トイレトレーニング項目は出さない。
- 保護者には「診断名」や「危険判定」を表示しない。
- 医師側には、出生直後からの便秘、胎便遅延、哺乳不良、体重増加不良、嘔吐、腹部膨満などの確認材料を整理する。
- 薬や処置については、保護者への自己判断を促さず、使用状況だけを記録する。

## 患者画面の質問案

### Q1. 最後にうんちが出たのはいつですか

選択肢:

- 今日
- 昨日
- 2-3日前
- 4日以上前
- わからない

医師側で見ること:

- 無排便期間
- 直近の変化
- 他の症状との組み合わせ

### Q2. うんちの様子はどれに近いですか

選択肢:

- やわらかい
- 普通
- 硬い
- コロコロ
- 水っぽい
- 細い便が続く
- わからない

医師側で見ること:

- 硬便、泥状便、水様便、細い便の有無
- 硬便と痛み、出しにくさの関係

### Q3. うんちのときの様子を教えてください

選択肢:

- 普段どおり
- いきむ
- 泣く、痛がる
- 顔を赤くして苦しそう
- 出そうで出ない
- 血がつくことがある

医師側で見ること:

- 排便時痛
- 裂肛を疑う情報
- 出しにくさの程度

### Q4. 便秘はいつごろから気になりますか

選択肢:

- 生まれてすぐから
- 生後数週間以内から
- ミルクや母乳の変化後から
- 離乳食を始めたころから
- 最近になってから
- わからない

医師側で見ること:

- 出生直後からの経過
- 食事変化、感染、生活変化との関係
- 早期発症の有無

### Q5. 出生時や1か月健診で異常を指摘されたことはありますか

選択肢:

- ない
- ある
- わからない

追加質問:

- 「ある」を選んだ場合、差し支えなければ内容を入力してください。

医師側で見ること:

- 出生時、1か月健診での指摘
- 便秘と関係する可能性がある既往や健診指摘
- 不明な場合は診察で追加確認

### Q6. ミルク・母乳・食事の様子を教えてください

選択肢:

- よく飲む、よく食べる
- 飲む量、食べる量が少ない
- 最近飲む量、食べる量が減った
- 離乳食を始めた、または内容が変わった
- 吐くことが増えた

医師側で見ること:

- 哺乳、食事量
- 離乳食との関係
- 嘔吐、摂取低下の有無

### Q7. ミルク、母乳、牛乳・乳製品との関係で気になることはありますか

選択肢:

- 特に気にならない
- ミルクを変えてから便が硬くなった気がする
- 牛乳・乳製品をとると便やおなかの調子が変わる気がする
- 母乳中で、母が牛乳・乳製品を多くとると便やおなかの調子が変わる気がする
- 湿疹、血便、吐き戻しなども気になる
- わからない

医師側で見ること:

- 牛乳・乳製品、ミルク変更、母乳中の母の食事との関連の訴え
- 湿疹、血便、嘔吐など、便秘以外の症状
- 牛乳アレルギーを診断するのではなく、医師が追加確認すべき情報として扱う

### Q8. おなかの様子で気になることはありますか

選択肢:

- 特に気にならない
- 張っているように見える
- 触ると嫌がる
- 吐くことがある
- 機嫌が悪い時間が増えた
- 発熱がある

医師側で見ること:

- 腹部膨満
- 嘔吐
- 全身状態
- 診察で追加確認すべき症状

### Q9. 体重や成長について気になることはありますか

選択肢:

- 特に気にならない
- 体重の増えが悪いと言われた
- 最近体重が増えにくい気がする
- 健診で相談するように言われた
- わからない

医師側で見ること:

- 体重増加不良
- 健診での指摘
- 発育評価の必要性

### Q10. 便秘に対して使っているものはありますか

選択肢:

- なし
- 医師から処方された薬
- 市販薬
- 浣腸
- 綿棒刺激
- その他

追加質問:

- どのくらいの頻度で使っていますか。
- 使った後、便は出ますか。
- 使うのをやめると悪化しますか。

医師側で見ること:

- 薬・処置の使用状況
- 処置依存の有無ではなく、診察で確認すべき使用頻度
- 自己判断で増減しないよう説明が必要か

### Q11. 最近2週間の記録

入力項目:

- 記録した日数
- うんちが出た日数
- 硬い便の日数
- 痛がった日数
- 吐いた日数
- 薬や処置を使った日数

医師側で見ること:

- 直近の実際の頻度
- 硬便、痛み、嘔吐、薬・処置使用の関係

## 条件分岐案

| 条件                                           | 追加で聞くこと                           | 医師側の扱い                                       |
| ---------------------------------------------- | ---------------------------------------- | -------------------------------------------------- |
| 生まれてすぐから、または生後数週間以内から便秘 | 出生時・1か月健診の指摘、哺乳、体重、腹部膨満、嘔吐 | 背景疾患を含め診察で確認すべき情報として上段に表示 |
| 出生時や1か月健診で指摘あり、または不明        | 指摘内容、健診での相談内容               | 出生時・健診情報の確認候補                         |
| 吐く、腹部膨満、哺乳低下、発熱、活気低下       | いつから、頻度、持続                     | 医師が診察で優先確認                               |
| 血がつく                                       | 便の硬さ、痛み、頻度                     | 裂肛などを含め診察で確認                           |
| 離乳食開始後に悪化                             | 食事内容、水分、便性状の変化             | 生活変化との関連として整理                         |
| ミルク変更、牛乳・乳製品、母の乳製品摂取と関連がありそう | 便性状、湿疹、血便、吐き戻し、摂取状況 | 牛乳アレルギーを含め、医師が追加確認する情報として整理 |
| 浣腸・綿棒刺激を頻回使用                       | 頻度、効果、処方の有無                   | 使用状況として整理し、自己判断を促さない           |

## 医師向けサマリー案

```text
【0-1歳 便秘サマリー】
年齢: 〇歳〇か月
最終排便: 〇日前
便性状: 〇〇
排便時の様子: 〇〇
発症時期: 〇〇
出生時・1か月健診の指摘: 〇〇
哺乳・食事: 〇〇
ミルク・牛乳・乳製品との関連: 〇〇
嘔吐・腹部膨満・発熱・活気: 〇〇
体重増加・健診指摘: 〇〇
薬・処置: 〇〇
直近日誌: 記録〇日、排便〇日、硬便〇日、痛み〇日、嘔吐〇日、薬・処置〇日

医師が追加確認する候補:
- 出生直後からの便秘か
- 出生時・1か月健診での指摘
- 哺乳量、体重増加、嘔吐、腹部膨満
- ミルク変更、牛乳・乳製品、母の乳製品摂取と症状の関係
- 肛門周囲所見
- 薬・浣腸・綿棒刺激の使い方
```

## 患者向けメモ案

```text
【便秘メモ】

今日のうんちの様子:
- 最後のうんち: 〇〇
- うんちの硬さ: 〇〇
- うんちのときの様子: 〇〇
- 飲み方・食べ方: 〇〇
- おなかや吐き気: 〇〇

次に相談すること:
- いつから便秘が気になるか
- うんちの硬さ、痛み、血がつくことがあるか
- ミルク、母乳、離乳食の変化
- 牛乳・乳製品やミルク変更で気になる変化
- 薬、浣腸、綿棒刺激を使った日

大事なこと:
- このメモでは薬の量を決めません。
- 薬や処置を増やす、減らす、やめる、再開することは診察で相談します。
```

## ChatGPT診察前整理モードに追加する観点

- 年齢が0-1歳であることを明示する。
- 出生直後または生後数週間以内からの便秘か。
- 出生時や1か月健診で指摘があったか。
- 哺乳低下、体重増加不良、嘔吐、腹部膨満、発熱、活気低下があるか。
- 離乳食開始やミルク変更など、便秘と同時期の変化があるか。
- 牛乳・乳製品、母の乳製品摂取、ミルク変更と便秘や湿疹、血便、吐き戻しの関連が疑われる訴えがあるか。
- 浣腸、綿棒刺激、市販薬の使用状況に不足がないか。

## ChatGPT治療方針検討モードに追加する観点

- 年齢が0-1歳であるため、背景疾患や全身状態の確認不足を明示する。
- 便秘薬の方針候補を出す前に、哺乳、体重、嘔吐、腹部膨満、発症時期、出生時・1か月健診の指摘の情報が十分かを確認する。
- 牛乳アレルギーを断定せず、牛乳・乳製品やミルク変更との関連があれば追加確認候補として整理する。
- 具体的な薬剤量、浣腸頻度、綿棒刺激頻度の指示は出させない。
- 保護者説明案は、自己判断ではなく、記録と診察相談につなげる文体にする。

## 実装前レビュー項目

- 0-1歳で患者画面に出してよい質問表現か。
- 出生時・1か月健診の指摘、嘔吐、腹部膨満、体重増加不良の扱いが強すぎないか、弱すぎないか。
- 牛乳アレルギー関連の質問が診断誘導や保護者の自己判断につながらないか。
- 綿棒刺激、浣腸、市販薬の聞き方が自己判断を促さないか。
- 日誌項目に「嘔吐日数」「薬・処置使用日数」を加えるべきか。
- 0歳前半と1歳台で質問をさらに分ける必要があるか。
- 医師向けサマリー上段に出すべき項目の優先順位。
