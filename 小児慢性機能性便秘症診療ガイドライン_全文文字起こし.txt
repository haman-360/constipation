第1章 定義・分類・診断基準
基本的事項
* 小児の便秘とは便が滞った、または便がでにくい状態である。
* 小児の便秘症とは「便秘」による身体症状が表れ、診療や治療を必要とする状態である。
* 小児の便秘症は病状の期間から慢性便秘症と一過性便秘症に、原因から機能性便秘症と器質性便秘症に分類される。
* 小児の慢性機能性便秘症の診断基準は、国際的に使用されている Rome IV 診断基準を使用する。
第2章 疫学
BQ2-1 小児慢性機能性便秘症の長期予後はどのようなものか？
1. 早期の診断と適切な治療介入により予後が改善する。（エビデンス：B 中等度 / 合意率：100%）
2. 治療の結果、症状が改善しても、再発することが多い。（エビデンス：C 弱い / 合意率：100%）
3. 慢性機能性便秘症の小児における生活の質（QOL）は、適切な治療により改善しうる。（エビデンス：C 弱い / 合意率：100%）
BQ2-2 小児慢性機能性便秘症の有病率はどれくらいか？

第4章 診断・検査
CQ4-1 便秘をきたす基礎疾患や病態の鑑別をどのように行うか？
1. 便秘をきたす疾患や病態として、Hirschsprung病や直腸肛門奇形などの外科的疾患や全身性の内科的疾患が存在するため、まずこれらの疾患を鑑別することを推奨する。（推奨：1 / エビデンス：C / 合意率：100%）
2. 便秘症をきたす基礎疾患を示唆する徴候（red flags）を認める場合は、基礎疾患除外のため精査することを推奨する。（推奨：1 / エビデンス：C / 合意率：100%）
CQ4-2 便塞栓（fecal impaction）の診断をどのように行うか？
1. 直腸肛門指診による診察を提案する。（推奨：2 / エビデンス：C / 合意率：96%）
2. 腹部超音波検査を推奨する。（推奨：1 / エビデンス：C / 合意率：100%）
3. 腹部X線検査を提案する。（推奨：2 / エビデンス：C / 合意率：96%）
CQ4-3 難治性便秘症（intractable constipation）の評価に検査は有用か？
1. 難治性便秘症の中で、器質的疾患の診断に至らなかった症例を鑑別するために、注腸造影検査を行うことを提案する。（推奨：2 / エビデンス：D / 合意率：100%）
2. 難治性便秘症の病態把握のために、大腸通過時間検査、大腸シンチグラフィー、大腸内圧検査を行うことを提案する。＜付帯事項＞上記の検査は、海外の専門医療機関からの報告がもとになっており、日本では一般的に行われている検査ではない。（推奨：2 / エビデンス：D / 合意率：100%）
第5章 治療総論
CQ5-1 どのような手順で小児慢性機能性便秘症を治療すべきか？
1. 便塞栓（fecal impaction）の有無を判定し、便塞栓のある患者においては、便塞栓除去（disimpaction）を行うことを推奨する。（推奨：1 / エビデンス：B / 合意率：100%）
2. 便塞栓除去後、もしくは便塞栓のない児では、維持療法を開始することを推奨する。（推奨：1 / エビデンス：B / 合意率：100%）
3. 標準的な維持療法で改善しない便秘症は、専門医療機関に紹介することを推奨する。（推奨：1 / エビデンス：C / 合意率：100%）
第6章 便塞栓除去（disimpaction）
CQ6-1 便塞栓除去（disimpaction）はどのように行うか？
1. 経口治療としてポリエチレングリコール（PEG）製剤を推奨する。（推奨：2 / エビデンス：C / 合意率：100%）
2. ポリエチレングリコール（PEG）製剤が投与できない患者には経肛門的治療を推奨する。（推奨：2 / エビデンス：C / 合意率：100%）
CQ6-2 便塞栓除去（disimpaction）の効果はどのように判定すべきか？
1. 便塞栓除去（disimpaction）の効果判定は、臨床症状・身体所見、排便回数、便形状、画像所見の変化を確認することを推奨する。（推奨：1 / エビデンス：C / 合意率：100%）
2. 便塞栓除去（disimpaction）の効果判定は、便塞栓除去開始日から1週間以内に、遅くとも2週間以内に行うことを推奨する。（推奨：1 / エビデンス：C / 合意率：100%）
